# 简介

青石是一个开源所见即所得Markdown编辑器，暂时只支持`mac-arm64`系统，使用[gfm](https%3A%2F%2Fgithub.github.com%2Fgfm%2F)语法，同时扩展了[Mermaid](https%3A%2F%2Fmermaid.js.org%2F)图形与[Katext](https%3A%2F%2Fkatex.org%2F)公式，支持明暗两种配色，目标为提高Markdown阅读与编辑体验。同时提供自由化的文档分享能力，让文档更易于分享传播，如将markdown同步至您自己的Linux服务器或者云存储。
